# 📱 Meu Website | iOS-Style Portfolio

Este repositório contém o código-fonte do meu portfólio pessoal. O design foi inteiramente concebido para simular a experiência visual e a fluidez do ecossistema **iOS**, utilizando componentes que remetem ao sistema operacional da Apple.

🔗 **Acesse online:** [luizpassaroni.com](https://www.luizpassaroni.com)

---

## 🎨 Design Philosophy
O objetivo deste projeto foi criar uma interface que unisse minimalismo e funcionalidade, seguindo as **Human Interface Guidelines (HIG)** da Apple:
- **Glassmorphism:** Uso de efeitos de desfoque (`backdrop-filter`) e transparências para criar profundidade.
- **SF Symbols & Typography:** Iconografia e fontes que respeitam a identidade visual da Apple.
- **Responsive Design:** Adaptável para iPhone, iPad e Desktop.
- **Dark Mode Support:** Suporte nativo ao tema escuro do sistema.

---

## 🛠️ Tecnologias Utilizadas
- **HTML5 & CSS3:** Estrutura e estilização customizada.
- **Tailwind CSS:** Framework utilitário para agilidade no layout e responsividade.
- **JavaScript (Vanilla):** Lógica de internacionalização (i18n) e interações de UI.
- **Cloudflare:** Gerenciamento de DNS e segurança (SSL/TLS).
- **GitHub Pages:** Hospedagem automatizada.

---

## 🌍 Funcionalidades
- **Internacionalização (i18n):** Suporte completo para Português (BR) e Inglês (US) com detecção automática do idioma do navegador.
- **Seletor de Idioma Customizado:** Componente estilo *iOS Picker*.
- **Timeline de Experiência:** Exibição cronológica de formação e carreira.
- **Hero Card:** Destaque dinâmico para projetos principais.
